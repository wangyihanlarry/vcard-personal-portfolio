# 项目上下文

## 项目目的
vCard个人作品集是一个完全响应式的个人作品集网站，展示个人的专业信息、技能、项目和联系方式。目标是为网页开发者和专业人士创建一个现代、优雅的在线形象。

## 技术栈
- HTML5（语义化标记）
- CSS3（自定义属性、弹性盒、网格）
- 原生JavaScript（ES6+）
- 无构建工具或框架 - 纯静态文件
- Google Fonts（Poppins字体）
- Ionicons图标库

## 项目约定

### 代码风格
- 使用语义化HTML5元素
- CSS自定义属性用于主题
- 类似BEM的CSS类命名约定
- ES6+ JavaScript特性
- 最小化依赖 - 优先使用原生解决方案

### 架构模式
- 静态站点架构（无服务器端处理）
- 基于组件的CSS组织
- 渐进增强方法
- 移动优先的响应式设计

### 测试策略
- 多浏览器手动测试
- 响应式设计测试
- 可访问性验证
- 使用Lighthouse进行性能测试

### Git工作流
- 主分支用于生产环境
- 功能分支用于新功能开发
- 约定式提交消息
- 拉取请求用于代码审查

## 领域背景
这是一个典型的网页开发者和设计师个人作品集网站。它包括：
- 个人信息和头像
- 联系方式
- 关于部分
- 简历/教育背景部分
- 作品集展示
- 博客部分
- 联系表单

## 重要约束
- 必须保持静态站点（无服务器端依赖）
- 需要跨浏览器兼容性
- 移动端响应式是强制性的
- 快速加载时间至关重要
- 需要符合可访问性标准

## 外部依赖
- Google Fonts API
- Ionicons CDN
- 无其他外部API或服务
